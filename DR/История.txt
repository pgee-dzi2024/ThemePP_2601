## Какво означава това
`openai-whisper` не чете WAV файла директно самостоятелно — той използва `ffmpeg` за зареждане и конвертиране на аудиото.  
Ако `ffmpeg` не е инсталиран или не е достъпен от командния ред, скриптът ще падне точно така.

## Дългосрочно решение
1. Инсталирай `ffmpeg` за Windows.
2. Добави папката с `ffmpeg.exe` в системния `PATH`.
3. Провери в терминал:
   ```bash
   ffmpeg -version
   ```
   Ако командата работи, Whisper ще може да зареди аудиото.

- проверява дали `ffmpeg` е наличен;

```python
import warnings
from pathlib import Path
import shutil
import sys

warnings.filterwarnings("ignore", category=UserWarning)

def main() -> int:
    audio_path = Path(__file__).parent / "audio" / "new.wav"

    if not audio_path.exists():

        return 1

    if shutil.which("ffmpeg") is None:
        print(

            "Whisper изисква ffmpeg, за да зареди аудиото.\n"
            "Инсталирай ffmpeg и добави ffmpeg.exe в PATH, после стартирай пак."
        )
        return 1

    print("Зареждане на Whisper large model (CPU)...")
    model = whisper.load_model("large", device="cpu")

    print("Транскрибиране на аудио...")
    result = model.transcribe(

        language="bg",
        task="transcribe",

    )
```

```text
C:\ffmpeg
```

```text
C:\ffmpeg\bin\ffmpeg.exe
C:\ffmpeg\bin\ffprobe.exe
```

### 3) Добави `ffmpeg` към PATH
1. Натисни **Start**
2. Потърси: **Environment Variables**
3. Отвори: **Edit the system environment variables**
4. Натисни **Environment Variables...**
5. В секцията **System variables** избери `Path`
6. Натисни **Edit**
7. Натисни **New**
8. Добави:

```text
C:\ffmpeg\bin
```

9. Натисни **OK** на всички прозорци

### 4) Провери инсталацията
Отвори нов `Command Prompt` или `PowerShell` и изпълни:

```bash
ffmpeg -version
```

Разбира се — **вариант 3** означава да държиш `ffmpeg.exe` **само в папката на проекта**, без да го инсталираш системно.

## Как работи идеята
Whisper търси `ffmpeg` чрез `PATH`.  
Ако не искаш да го добавяш в системните променливи, можеш:

1. да сложиш `ffmpeg.exe` в папка вътре в проекта;
2. да добавиш тази папка **временно** към `PATH` само за текущото стартиране на Python скрипта.

Така:
- не пипаш Windows настройките;
- работи само за този проект;
- после, ако искаш, можеш да го махнеш лесно.

---

## Стъпка по стъпка

### 1) Намери `ffmpeg.exe`
От разархивираната папка, която вече имаш в `Downloads`, потърси:

```text
ffmpeg.exe
ffprobe.exe
```

Обикновено са в:

```text
...\bin\ffmpeg.exe
```

---

### 2) Копирай цялата `bin` папка в проекта
Най-удобно е да направиш примерно така:

```text
Theme_01_Ivan/
  work/
    test_whisper/
      audio/
      tools/
        ffmpeg/
          bin/
            ffmpeg.exe
            ffprobe.exe
      whisper_test.py
```

Или просто:

```text
Theme_01_Ivan/work/test_whisper/ffmpeg/bin/ffmpeg.exe
```

Важно е **`ffmpeg.exe` да е в папка, която знаеш точно къде е**.

---

### 3) Добави тази папка временно към PATH в кода
Това става **преди** да извикаш Whisper.

Ето как да го направиш в `whisper_test.py`:

```python
import warnings
import os
from pathlib import Path
import shutil

warnings.filterwarnings("ignore", category=UserWarning)

import whisper


def main() -> int:
    audio_path = Path(__file__).parent / "audio" / "new.wav"
    ffmpeg_dir = Path(__file__).parent / "ffmpeg" / "bin"

    if not audio_path.exists():
        print(f"Грешка: аудиофайлът не е намерен: {audio_path}")
        return 1

    if not ffmpeg_dir.exists():
        print(f"Грешка: папката с ffmpeg не е намерена: {ffmpeg_dir}")
        return 1

    os.environ["PATH"] = str(ffmpeg_dir) + os.pathsep + os.environ.get("PATH", "")

    if shutil.which("ffmpeg") is None:
        print("Грешка: ffmpeg пак не е открит, провери дали ffmpeg.exe е вътре в папката bin.")
        return 1

    print("Зареждане на Whisper large model (CPU)...")
    model = whisper.load_model("large", device="cpu")

    print("Транскрибиране на аудио...")
    result = model.transcribe(
        str(audio_path),
        language="bg",
        task="transcribe",
        beam_size=1,
        temperature=0.0,
        initial_prompt="Това е разговор на български език. Моля, транскрибирай внимателно думите.",
        compression_ratio_threshold=2.2,
        logprob_threshold=-0.6,
        no_speech_threshold=0.5,
        condition_on_previous_text=False,
        word_timestamps=False,
        fp16=False,
    )

    print("\n--- TRANSCRIPTION ---")
    print(result["text"])
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
```

---

## Какво точно прави този код
- `ffmpeg_dir = Path(__file__).parent / "ffmpeg" / "bin"`  
  → очаква `ffmpeg.exe` да е в папка `ffmpeg/bin` до скрипта.

- `os.environ["PATH"] = ...`  
  → добавя тази папка само за текущото изпълнение на програмата.

- `shutil.which("ffmpeg")`  
  → проверява дали Python вече може да намери `ffmpeg`.

---

## Ако искаш още по-лесен вариант
Можеш да не променяш кода, а да стартираш скрипта от командния ред така:

```bash
set PATH=D:\DZI_2026\PP\Theme_01_Ivan\work\test_whisper\ffmpeg\bin;%PATH%
python whisper_test.py
```

Това важи **само за този терминален прозорец**.

---

## Кое е по-добро
- **За тест**: локална папка + временно PATH
- **За дългосрочно**: системен PATH

---

Ако искаш, мога да ти кажа **точно къде да сложиш `ffmpeg.exe` в твоя проект**, като използвам текущата структура на папките ти.

```python
import warnings
import os
from pathlib import Path
import shutil

warnings.filterwarnings("ignore", category=UserWarning)

import whisper


def main() -> int:

    audio_path = Path(__file__).parent / "audio" / "new.wav"
    ffmpeg_dir = Path(__file__).parent / "ffmpeg" / "bin"

    if not audio_path.exists():
        print(f"Грешка: аудиофайлът не е намерен: {audio_path}")
        return 1

    if not ffmpeg_dir.exists():
        print(f"Грешка: папката с ffmpeg не е намерена: {ffmpeg_dir}")
        return 1

    os.environ["PATH"] = str(ffmpeg_dir) + os.pathsep + os.environ.get("PATH", "")

    if shutil.which("ffmpeg") is None:
        print("Грешка: ffmpeg пак не е открит, провери дали ffmpeg.exe е вътре в папката bin.")
        return 1

    print("Зареждане на Whisper large model (CPU)...")
    model = whisper.load_model("large", device="cpu")

    print("Транскрибиране на аудио...")
    result = model.transcribe(
        str(audio_path),
        language="bg",
        task="transcribe",
        beam_size=1,
        temperature=0.0,
        initial_prompt="Това е разговор на български език. Моля, транскрибирай внимателно думите.",
        compression_ratio_threshold=2.2,
        logprob_threshold=-0.6,
        no_speech_threshold=0.5,
        condition_on_previous_text=False,
        word_timestamps=False,
        fp16=False,
    )

    print("\n--- TRANSCRIPTION ---")
    print(result["text"])
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
```

# Как да реализираме асинхронността без сложна инфраструктура

Има два лесни начина:

## Вариант A: `ThreadPoolExecutor`
- много подходящ за локален проект;
- без Celery, без Redis, без допълнителни услуги;
- достатъчно добър за Whisper обработка.

## Вариант B: `asyncio`
- не е най-подходящо тук, защото Whisper е CPU-bound и блокиращ;
- за този случай **не е най-добрият избор**.

### Препоръка
Избери **`ThreadPoolExecutor`**.  
Това е най-логичното решение за минимално асинхронно приложение.

# 1) Нов файл: `main/services/transcription_service.py`

Тук ще сложим:
- логиката за Whisper;
- генериране на SRT/TXT;
- работа с `ffmpeg` в `PATH`;
- helper за обработка на качения файл.

```python
from __future__ import annotations

import os
import shutil
import subprocess
from pathlib import Path
from typing import Any, Dict, List

import whisper


def ensure_ffmpeg_available() -> None:
    if shutil.which("ffmpeg") is None:
        raise RuntimeError(
            "ffmpeg не е намерен в PATH. Whisper изисква ffmpeg за обработка на аудиото."
        )


def load_whisper_model():
    ensure_ffmpeg_available()
    return whisper.load_model("large", device="cpu")


def format_srt_timestamp(seconds: float) -> str:
    milliseconds = int(round(seconds * 1000))
    hours, remainder = divmod(milliseconds, 3_600_000)
    minutes, remainder = divmod(remainder, 60_000)
    secs, ms = divmod(remainder, 1000)
    return f"{hours:02}:{minutes:02}:{secs:02},{ms:03}"


def build_srt(segments: List[Dict[str, Any]]) -> str:
    lines: list[str] = []
    for index, segment in enumerate(segments, start=1):
        start = format_srt_timestamp(float(segment["start"]))
        end = format_srt_timestamp(float(segment["end"]))
        text = str(segment["text"]).strip()
        lines.append(f"{index}\n{start} --> {end}\n{text}\n")
    return "\n".join(lines).strip() + "\n"


def build_txt(segments: List[Dict[str, Any]]) -> str:
    return "\n".join(str(segment["text"]).strip() for segment in segments).strip() + "\n"


def transcribe_media_file(file_path: Path) -> Dict[str, Any]:
    model = load_whisper_model()

    result = model.transcribe(
        str(file_path),
        language="bg",
        task="transcribe",
        beam_size=1,
        temperature=0.0,
        initial_prompt="Това е разговор на български език. Моля, транскрибирай внимателно думите.",
        compression_ratio_threshold=2.2,
        logprob_threshold=-0.6,
        no_speech_threshold=0.5,
        condition_on_previous_text=False,
        word_timestamps=False,
        fp16=False,
    )

    segments = result.get("segments", [])
    text = str(result.get("text", "")).strip()
    srt = build_srt(segments)
    txt = build_txt(segments)

    return {
        "text": text,
        "srt": srt,
        "txt": txt,
    }
```

# 2) Нов файл: `main/services/task_manager.py`

Тук ще държим задачите в паметта и ще ги изпълняваме с `ThreadPoolExecutor`.

```python
from __future__ import annotations

import uuid
from concurrent.futures import ThreadPoolExecutor
from pathlib import Path
from typing import Any, Dict

from .transcription_service import transcribe_media_file

EXECUTOR = ThreadPoolExecutor(max_workers=2)
TASKS: dict[str, dict[str, Any]] = {}


def create_task(file_path: Path) -> str:
    task_id = uuid.uuid4().hex

    TASKS[task_id] = {
        "status": "pending",
        "result": None,
        "error": None,
    }

    future = EXECUTOR.submit(_run_task, task_id, file_path)
    TASKS[task_id]["future"] = future
    return task_id


def _run_task(task_id: str, file_path: Path) -> None:
    try:
        TASKS[task_id]["status"] = "processing"
        result = transcribe_media_file(file_path)
        TASKS[task_id]["status"] = "done"
        TASKS[task_id]["result"] = result
    except Exception as exc:
        TASKS[task_id]["status"] = "error"
        TASKS[task_id]["error"] = str(exc)
    finally:
        TASKS[task_id].pop("future", None)


def get_task(task_id: str) -> dict[str, Any] | None:
    return TASKS.get(task_id)
```

Тук ще направим двата основни API endpoint-а.

```python
from __future__ import annotations

from pathlib import Path

from django.conf import settings
from django.core.files.storage import default_storage
from django.core.files.base import ContentFile
from rest_framework import status
from rest_framework.response import Response
from rest_framework.views import APIView

from .serializers import MediaUploadSerializer
from .services.task_manager import create_task, get_task


class TranscribeCreateAPIView(APIView):
    def post(self, request):
        serializer = MediaUploadSerializer(data=request.data)
        serializer.is_valid(raise_exception=True)

        uploaded_file = serializer.validated_data["media_file"]
        media_dir = Path(settings.MEDIA_ROOT) / "uploads"
        media_dir.mkdir(parents=True, exist_ok=True)

        saved_path = media_dir / uploaded_file.name
        with default_storage.open(str(saved_path), "wb") as destination:
            for chunk in uploaded_file.chunks():
                destination.write(chunk)

        task_id = create_task(saved_path)

        return Response(
            {
                "task_id": task_id,
                "status": "pending",
            },
            status=status.HTTP_202_ACCEPTED,
        )


class TranscribeStatusAPIView(APIView):
    def get(self, request, task_id: str):
        task = get_task(task_id)
        if task is None:
            return Response(
                {"detail": "Task not found."},
                status=status.HTTP_404_NOT_FOUND,
            )

        payload = {
            "task_id": task_id,
            "status": task["status"],
        }

        if task["status"] == "done" and task["result"]:
            payload.update(task["result"])

        if task["status"] == "error":
            payload["error"] = task["error"]

        return Response(payload)
```

## Какво да направиш в браузъра
В DRF browsable API при `multipart/form-data` трябва да имаш:

- поле **`media_file`** или
- бутон **Choose File / Избери файл**

и там да посочиш:

```text
dp/main/test/new.wav
```

или да го избереш ръчно от компютъра.

---

## Важното
Ако виждаш само поле **Content** и няма поле за файл, значи API-то още **не приема multipart upload правилно** от browsable interface.

Това обикновено се оправя като в API view-а се добавят parser-и за multipart upload.

### Какво трябва да имаш в backend-а
В DRF view-а за качване е добре да има:

- `MultiPartParser`
- `FormParser`

Така браузърът ще покаже нормално поле за файл, вместо само текстово поле.

---

## Тоест:
### В `content` не пишеш нищо.
Там **не се въвежда файлът ръчно**.

### Правилният начин е:
1. Отвори endpoint-а в браузъра
2. Избери `multipart/form-data`
3. Натисни **Choose File**
4. Избери `new.wav`
5. Натисни **POST**

```python
from __future__ import annotations

from pathlib import Path

from django.conf import settings
from django.core.files.storage import default_storage
from rest_framework import status
from rest_framework.parsers import FormParser, MultiPartParser
from rest_framework.response import Response
from rest_framework.views import APIView

from .serializers import MediaUploadSerializer
from .services.task_manager import create_task, get_task


class TranscribeCreateAPIView(APIView):
    parser_classes = [MultiPartParser, FormParser]

    def post(self, request):
        serializer = MediaUploadSerializer(data=request.data)
        serializer.is_valid(raise_exception=True)

        uploaded_file = serializer.validated_data["media_file"]
        media_dir = Path(settings.MEDIA_ROOT) / "uploads"
        media_dir.mkdir(parents=True, exist_ok=True)

        saved_path = media_dir / uploaded_file.name
        with default_storage.open(str(saved_path), "wb") as destination:
            for chunk in uploaded_file.chunks():
                destination.write(chunk)

        task_id = create_task(saved_path)

        return Response(
            {
                "task_id": task_id,
                "status": "pending",
            },
            status=status.HTTP_202_ACCEPTED,
        )


class TranscribeStatusAPIView(APIView):
    def get(self, request, task_id: str):
        task = get_task(task_id)
        if task is None:
            return Response(
                {"detail": "Task not found."},
                status=status.HTTP_404_NOT_FOUND,
            )

        payload = {
            "task_id": task_id,
            "status": task["status"],
        }

        if task["status"] == "done" and task["result"]:
            payload.update(task["result"])

        if task["status"] == "error":
            payload["error"] = task["error"]

        return Response(payload)
```

- можеш да избираш директно `new.wav`

```python
from __future__ import annotations

import os
import shutil
import subprocess
from pathlib import Path
from typing import Any, Dict, List

import whisper


def ensure_ffmpeg_available() -> None:
    if shutil.which("ffmpeg") is None:
        raise RuntimeError(
            "ffmpeg не е намерен в PATH. Whisper изисква ffmpeg за обработка на аудиото."
        )


def load_whisper_model():
    ensure_ffmpeg_available()
    return whisper.load_model("large", device="cpu")


def format_srt_timestamp(seconds: float) -> str:
    milliseconds = int(round(seconds * 1000))
    hours, remainder = divmod(milliseconds, 3_600_000)
    minutes, remainder = divmod(remainder, 60_000)
    secs, ms = divmod(remainder, 1000)
    return f"{hours:02}:{minutes:02}:{secs:02},{ms:03}"


def build_srt(segments: List[Dict[str, Any]]) -> str:
    lines: list[str] = []
    for index, segment in enumerate(segments, start=1):
        start = format_srt_timestamp(float(segment["start"]))
        end = format_srt_timestamp(float(segment["end"]))
        text = str(segment["text"]).strip()
        lines.append(f"{index}\n{start} --> {end}\n{text}\n")
    return "\n".join(lines).strip() + "\n"


def build_txt(segments: List[Dict[str, Any]]) -> str:
    return "\n".join(str(segment["text"]).strip() for segment in segments).strip() + "\n"


def transcribe_media_file(file_path: Path) -> Dict[str, Any]:
    model = load_whisper_model()

    result = model.transcribe(
        str(file_path),
        language="bg",
        task="transcribe",
        beam_size=1,
        temperature=0.0,
        initial_prompt="Това е разговор на български език. Моля, транскрибирай внимателно думите.",
        compression_ratio_threshold=2.2,
        logprob_threshold=-0.6,
        no_speech_threshold=0.5,
        condition_on_previous_text=False,
        word_timestamps=False,
        fp16=False,
    )

    segments = result.get("segments", [])
    text = str(result.get("text", "")).strip()
    srt = build_srt(segments)
    txt = build_txt(segments)

    return {
        "text": text,
        "srt": srt,
        "txt": txt,
    }
```

```python
from __future__ import annotations

from pathlib import Path

from django.conf import settings
from django.core.files.storage import default_storage
from rest_framework import status
from rest_framework.generics import GenericAPIView
from rest_framework.parsers import FormParser, MultiPartParser
from rest_framework.response import Response

from .serializers import MediaUploadSerializer
from .services.task_manager import create_task, get_task


class TranscribeCreateAPIView(GenericAPIView):
    serializer_class = MediaUploadSerializer
    parser_classes = [MultiPartParser, FormParser]

    def post(self, request, *args, **kwargs):
        serializer = self.get_serializer(data=request.data)
        serializer.is_valid(raise_exception=True)

        uploaded_file = serializer.validated_data["media_file"]
        media_dir = Path(settings.MEDIA_ROOT) / "uploads"
        media_dir.mkdir(parents=True, exist_ok=True)

        saved_path = media_dir / uploaded_file.name
        with default_storage.open(str(saved_path), "wb") as destination:
            for chunk in uploaded_file.chunks():
                destination.write(chunk)

        task_id = create_task(saved_path)

        return Response(
            {
                "task_id": task_id,
                "status": "pending",
            },
            status=status.HTTP_202_ACCEPTED,
        )


class TranscribeStatusAPIView(GenericAPIView):
    def get(self, request, task_id: str, *args, **kwargs):
        task = get_task(task_id)
        if task is None:
            return Response(
                {"detail": "Task not found."},
                status=status.HTTP_404_NOT_FOUND,
            )

        payload = {
            "task_id": task_id,
            "status": task["status"],
        }

        if task["status"] == "done" and task["result"]:
            payload.update(task["result"])

        if task["status"] == "error":
            payload["error"] = task["error"]

        return Response(payload)
```

```html
        const { createApp } = Vue;

        createApp({
            data() {
                return {
                    selectedFile: null,
                    taskId: null,
                    taskStatus: '',
                    isUploading: false,
                    isPolling: false,
                    errorMessage: '',
                    subtitles: '',
                    srtText: '',
                    txtText: '',
                    pollingTimer: null
                };
            },
            methods: {
                onFileChange(event) {
                    this.selectedFile = event.target.files[0] || null;
                    this.errorMessage = '';
                },

                async uploadFile() {
                    if (!this.selectedFile) {
                        this.errorMessage = 'Моля, изберете файл.';
                        return;
                    }

                    this.errorMessage = '';
                    this.subtitles = '';
                    this.srtText = '';
                    this.txtText = '';
                    this.taskStatus = '';
                    this.isUploading = true;

                    try {
                        const formData = new FormData();
                        formData.append('media_file', this.selectedFile);

                        const response = await axios.post('/api/transcribe/', formData, {
                            headers: {
                                'Content-Type': 'multipart/form-data'
                            }
                        });

                        this.taskId = response.data.task_id;
                        this.taskStatus = response.data.status;
                        this.isUploading = false;
                        this.isPolling = true;
                        this.startPolling();
                    } catch (error) {
                        this.isUploading = false;
                        this.errorMessage = error?.response?.data?.detail || 'Възникна грешка при качването на файла.';
                    }
                },

                startPolling() {
                    if (this.pollingTimer) {
                        clearInterval(this.pollingTimer);
                    }

                    this.pollingTimer = setInterval(async () => {
                        try {
                            const response = await axios.get(`/api/transcribe/${this.taskId}/`);
                            this.taskStatus = response.data.status;

                            if (response.data.status === 'done') {
                                this.subtitles = response.data.text || '';
                                this.srtText = response.data.srt || '';
                                this.txtText = response.data.txt || '';
                                this.isPolling = false;
                                clearInterval(this.pollingTimer);
                                this.pollingTimer = null;
                            }

                            if (response.data.status === 'error') {
                                this.errorMessage = response.data.error || 'Възникна грешка при обработката.';
                                this.isPolling = false;
                                clearInterval(this.pollingTimer);
                                this.pollingTimer = null;
                            }
                        } catch (error) {
                            this.errorMessage = 'Грешка при проверка на статуса на задачата.';
                            this.isPolling = false;
                            clearInterval(this.pollingTimer);
                            this.pollingTimer = null;
                        }
                    }, 2000);
                },

                downloadFile(content, filename, mimeType) {
                    if (!content) {
                        return;
                    }
                    const blob = new Blob([content], { type: mimeType });
                    const url = URL.createObjectURL(blob);
                    const link = document.createElement('a');
                    link.href = url;
                    link.download = filename;
                    document.body.appendChild(link);
                    link.click();
                    document.body.removeChild(link);
                    URL.revokeObjectURL(url);
                },

                downloadSrt() {
                    this.downloadFile(this.srtText, 'subtitles.srt', 'text/plain');
                },

                downloadTxt() {
                    this.downloadFile(this.txtText, 'subtitles.txt', 'text/plain');
                }
            },
            beforeUnmount() {
                if (this.pollingTimer) {
                    clearInterval(this.pollingTimer);
                }
            }
        }).mount('#app');
```

```python
from __future__ import annotations

import os
import shutil
import subprocess
from pathlib import Path
from typing import Any, Dict, List

import whisper


def ensure_ffmpeg_available() -> None:
    if shutil.which("ffmpeg") is None:
        raise RuntimeError(
            "ffmpeg не е намерен в PATH. Whisper изисква ffmpeg за обработка на аудиото."
        )


def load_whisper_model():
    ensure_ffmpeg_available()
    return whisper.load_model("large", device="cpu")


def format_srt_timestamp(seconds: float) -> str:
    milliseconds = int(round(seconds * 1000))
    hours, remainder = divmod(milliseconds, 3_600_000)
    minutes, remainder = divmod(remainder, 60_000)
    secs, ms = divmod(remainder, 1000)
    return f"{hours:02}:{minutes:02}:{secs:02},{ms:03}"


def build_srt(segments: List[Dict[str, Any]]) -> str:
    lines: list[str] = []
    for index, segment in enumerate(segments, start=1):
        start = format_srt_timestamp(float(segment["start"]))
        end = format_srt_timestamp(float(segment["end"]))
        text = str(segment["text"]).strip()
        lines.append(f"{index}\n{start} --> {end}\n{text}\n")
    return "\n".join(lines).strip() + "\n"


def build_txt(segments: List[Dict[str, Any]]) -> str:
    return "\n".join(str(segment["text"]).strip() for segment in segments).strip() + "\n"


def transcribe_media_file(file_path: Path) -> Dict[str, Any]:
    model = load_whisper_model()

    result = model.transcribe(
        str(file_path),
        language="bg",
        task="transcribe",
        beam_size=1,
        temperature=0.0,
        initial_prompt="Това е разговор на български език. Моля, транскрибирай внимателно думите.",
        compression_ratio_threshold=2.2,
        logprob_threshold=-0.6,
        no_speech_threshold=0.5,
        condition_on_previous_text=False,
        word_timestamps=False,
        fp16=False,
    )

    segments = result.get("segments", [])
    text = str(result.get("text", "")).strip()
    srt = build_srt(segments)
    txt = build_txt(segments)

    return {
        "text": text,
        "srt": srt,
        "txt": txt,
    }
```

```python
def ensure_ffmpeg_available() -> None:
    if FFMPEG_DIR.exists():
        os.environ["PATH"] = str(FFMPEG_DIR) + os.pathsep + os.environ.get("PATH", "")

    if shutil.which("ffmpeg") is None:
        raise RuntimeError(
            f"ffmpeg не е намерен в PATH. Потърсен е и в локалната папка: {FFMPEG_DIR}"
        )


def load_whisper_model():
    ensure_ffmpeg_available()
    return whisper.load_model("large", device="cpu")


def format_srt_timestamp(seconds: float) -> str:
    milliseconds = int(round(seconds * 1000))
    hours, remainder = divmod(milliseconds, 3_600_000)
    minutes, remainder = divmod(remainder, 60_000)
    secs, ms = divmod(remainder, 1000)
    return f"{hours:02}:{minutes:02}:{secs:02},{ms:03}"


def build_srt(segments: List[Dict[str, Any]]) -> str:
    lines: list[str] = []
    for index, segment in enumerate(segments, start=1):
        start = format_srt_timestamp(float(segment["start"]))
        end = format_srt_timestamp(float(segment["end"]))
        text = str(segment["text"]).strip()
        lines.append(f"{index}\n{start} --> {end}\n{text}\n")
    return "\n".join(lines).strip() + "\n"


def build_txt(segments: List[Dict[str, Any]]) -> str:
    return "\n".join(str(segment["text"]).strip() for segment in segments).strip() + "\n"


def transcribe_media_file(file_path: Path) -> Dict[str, Any]:
    model = load_whisper_model()

    result = model.transcribe(
        str(file_path),
        language="bg",
        task="transcribe",
        beam_size=1,
        temperature=0.0,
        initial_prompt="Това е разговор на български език. Моля, транскрибирай внимателно думите.",
        compression_ratio_threshold=2.2,
        logprob_threshold=-0.6,
        no_speech_threshold=0.5,
        condition_on_previous_text=False,
        word_timestamps=False,
        fp16=False,
    )

    segments = result.get("segments", [])
    text = str(result.get("text", "")).strip()
    srt = build_srt(segments)
    txt = build_txt(segments)

    return {
        "text": text,
        "srt": srt,
        "txt": txt,
    }
```
